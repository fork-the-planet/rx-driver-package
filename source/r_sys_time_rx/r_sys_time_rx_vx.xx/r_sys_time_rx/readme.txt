r_sys_time_rx
========

Overview
--------------------------------------------------------------------------------
The r_sys_time_rx module has a system timer structure and simple-scheduler.
Configuration options can be found in "\r_config\r_sys_time_rx_config.h".


Features
--------
* All RX MCUs are supported.
