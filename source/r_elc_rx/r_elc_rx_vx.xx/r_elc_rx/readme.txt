r_elc_rx
=======

Overview
--------------------------------------------------------------------------------
The r_elc_rx module can be used to control the on board event link controller
for the target device.
The event link controller (ELC) connects (links) the events generated by various 
peripheral modules to different modules. Event linking allows direct cooperation 
between the modules without CPU intervention.

A configuration option for this exists in "r_config\r_elc_rx_config.h".


Features
--------
* Supports Event control, timer event input control, Port event I/O control
